>Problema 3

La siguiente secuencia iterativa esta definida por las siguientes reglas sobre números positivos:

n = n/2 (si n es par)
n = 3n + 1 (si n es inpar)
si n = 1 termina la secuencia

Usando estas reglas y empezando con el número 13, podemos generar la siguiente secuencia:

13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1

Se puede observar que esta secuencia contiene 10 términos contando el primero y el ultimo.
¿Qué número inicial, por debajo de 1 millón, produce la secuencia mas larga?

Nota:una vez que la secuencia inicia el numero puede superar el millón.
